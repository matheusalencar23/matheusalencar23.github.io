Experiências

Veja um pouco sobre as tecnologias ferramentas que impulsionaram minha trajetória como programador. Desde linguagens de programação até frameworks e softwares especializados, cada ferramenta moldou minha habilidade técnica e minha abordagem criativa para resolver problemas complexos.

Desenvolvimento Front End

O desenvolvimento front-end é minha principal afinidade. Trabalho na criação de interfaces que os usuários interagem, utilizando tecnologias como HTML, CSS, JavaScript e TypeScript. Cada projeto é uma oportunidade para combinar habilidades técnicas com criatividade, visando proporcionar a melhor experiência possível ao usuário final.

Angular

Angular é uma estrutura poderosa para desenvolvimento front-end, ideal para criar aplicativos da web dinâmicos e interativos. Com recursos avançados como vinculação de dados bidirecional e injeção de dependência, Angular permite construir interfaces robustas e escaláveis. Sua arquitetura modular e grande comunidade de desenvolvedores o tornam uma escolha popular para projetos de grande porte.

React

React é uma biblioteca JavaScript para construir interfaces de usuário interativas e reativas. Sua abordagem baseada em componentes permite o desenvolvimento modular e escalável de aplicativos da web. Com renderização eficiente do lado do cliente, React oferece uma experiência de usuário rápida e fluida, sendo uma escolha preferida para interfaces modernas e dinâmicas.

Sobre mim

Desenvolvedor Front End

Olá! Sou um desenvolvedor front-end com experiência em tecnologias como Angular e React.
Graduei-me em Engenharia Mecânica pela Universidade Federal do Ceará (UFC) em 2021, onde obtive uma sólida formação em matemática, lógica e resolução de problemas. Durante minha formação, tive a oportunidade de estudar linguagens como C e Python, que me proporcionaram uma base sólida em programação estruturada e desenvolvimento de algoritmos eficientes.
Após me identificar com o universo da programação, concentrei meus esforços em aprender tecnologias front-end modernas, como Angular e React, que são amplamente utilizadas. Essas frameworks me permitem criar interfaces de usuário interativas e responsivas.
Além disso, tenho conhecimento em outras tecnologias relacionadas ao desenvolvimento front-end, como HTML, CSS, JavaScript e TypeScript. Também estou familiarizado com ferramentas de teste e pré-processadores CSS, como Jest, Karma, Jasmine, Sass e Less. Essas habilidades combinadas me permitem criar interfaces atraentes e funcionais, enquanto também garanto a qualidade e a manutenibilidade do código.
Tenho plena capacidade de colaborar efetivamente em equipes multidisciplinares. Trabalhando em conjunto com designers, desenvolvedores de back-end e stakeholders.
Continuo me atualizando constantemente sobre as últimas tendências e avanços no desenvolvimento front-end. Tenho grande interesse em explorar novas tecnologias, como Vue.js, Solid.js e Svelte, além de tecnologias de desenvolvimento mobile.
Atualmente, estou cursando pós-graduação em Engenharia de Software pela PUC Minas, onde tenho aprofundado meus conhecimentos em desenvolvimento de software e práticas avançadas de programação.
