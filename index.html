Sobre mim

Desenvolvedor Front End

Sou um desenvolvedor de software com sólidos conhecimentos em tecnologias para desenvolvimento frontend, incluindo Angular, React, HTML, CSS, JavaScript e TypeScript. Tenho experiência com pré-processadores CSS como Sass e Less, e ferramentas de estilização como Bootstrap e Tailwind. Também trabalho com bibliotecas de componentes, como Material e Prime, e ferramentas de teste, como Jest, Karma e Jasmine.
Além disso, tenho familiaridade com tecnologias usadas para outras finalidades, como Java e Spring Boot para a construção de APIs e serviços back-end, e Python para trabalho com dados, criação de relatórios dinâmicos. Além de experiência com Cypress para automação de testes.
Tenho também conhecimento em bancos de dados relacionais, como MySQL, PostgreSQL e SQL Server, e bancos não relacionais, como MongoDB.
